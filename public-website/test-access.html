🌙 Nouveau : Système de thème

Le site supporte maintenant 3 modes de thème :

  • Mode Clair - Thème par défaut avec fond blanc
  • Mode Sombre - Thème sombre avec fond noir
  • Mode Système - Suit automatiquement les préférences du système

Comment tester :

  1. Cherchez l'icône ☀️/🌙 dans le header (à côté du sélecteur de langue)
  2. Cliquez dessus pour ouvrir le menu de sélection de thème
  3. Testez les 3 options : Clair, Sombre, Système
  4. Le thème est sauvegardé automatiquement dans le navigateur

🔧 Fonctionnalités de l'accès à l'application

Quand vous cliquez sur le bouton "Application" ou "Se connecter" dans le header :

  1. Vous êtes redirigé vers /[locale]/app
  2. Le système vérifie si vous êtes déjà connecté à l'application interne
  3. Si vous êtes connecté : accès direct au dashboard
  4. Si vous n'êtes pas connecté : redirection vers la page de connexion
  5. L'URL de l'application interne est configurée via NEXT_PUBLIC_INTERNAL_APP_URL

⚙️ Configuration requise

Pour que l'accès à l'application fonctionne complètement, assurez-vous que :

  • La variable d'environnement NEXT_PUBLIC_INTERNAL_APP_URL pointe vers votre application principale
  • L'application principale a les endpoints d'API nécessaires (/api/auth/verify-session, /api/auth/logout)
  • Les cookies de session sont partagés entre les deux applications
